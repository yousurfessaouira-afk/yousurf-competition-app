# YouSurf Competition App

Mobile-first, dark-mode competition management system for watersports events.

The current GitHub Pages-compatible app is pure HTML/CSS/JS and stores event data locally in the browser for offline-first operation. It includes:

- Event creation and admin dashboard
- Division management and rider database
- Heat and round setup with automatic draw helpers
- Multi-judge scoring sheets
- Live ranking engine using averaged judge totals
- Head judge validation and result locking
- Automatic heat progression for advancing riders
- Beach marshal rider check-in screen
- Speaker notes and full-screen live display mode

Open `index.html` directly or publish the repository with GitHub Pages.
